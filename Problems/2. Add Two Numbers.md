https://leetcode.cn/problems/add-two-numbers
***
*You are given two non-empty linked lists representing two non-negative integers. The digits are stored in reverse order, and each of their nodes contains a single digit. Add the two numbers and return the sumÂ as a linked list.*

*You may assume the two numbers do not contain any leading zero, except the number 0 itself.*
***
```
class ListNode(object):
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next
        
class Solution(object):
    def addTwoNumbers(self, l1, l2):
        """
        :type l1: ListNode
        :type l2: ListNode
        :rtype: ListNode
        """
        head = curr = ListNode(0)
        carry = 0
        num1 = 0
        num2 = 0
        while l1 or l2 or carry:
            if l1:
                num1 = l1.val
                l1 = l1.next
            else:
                num1 = 0
            if l2:
                num2 = l2.val
                l2 = l2.next
            else:
                num2 = 0
            sum = num1 + num2 + carry
            carry = sum // 10
            curr.next = ListNode(sum % 10)
            curr = curr.next
        return head.next
```
